# Contributing results

Contributions of benchmark results are welcome!
See the
[benchmark README](https://github.com/Stella-AI/stella/blob/main/benchmark/README.md)
for information on running stella's code editing benchmarks.
Submit results by opening a PR with edits to the
[benchmark results data files](https://github.com/Stella-AI/stella/blob/main/stella/website/_data/).
